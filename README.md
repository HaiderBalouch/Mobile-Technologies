Tutorial and Lab Work completed for the Mobile Technologies at the University of Canberra.
Each week is packaged with as a .zip file containing the full Android Studio Project for that session.

Tech Stack:
Language: Java/Kotlin
IDE: Android Studio
Platform: Android
Services: Firebase Realtime Database, Google ML Kit, Google Maps & Places API, SQLite

Getting Started:
To run any project you will need to -
1. Download and extract the .zip file for the week you want
2. Open the extracted folder in Android Studio
3. Let Gradle sync complete
4. Run the app on the emulator or physical Android device (Some projects require Firebase or Google Maps API keys to function)

Some notes to consider:
- This repo is for educational purposes as part of university coursework
- Projects are standalone per week and are not dependent on eachother
- Week 1 had no computer lab session, only lecture content and installation setup
